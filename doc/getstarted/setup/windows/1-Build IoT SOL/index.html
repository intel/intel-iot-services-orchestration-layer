Install Intel IoT SOL

Download and install IoT SOL by following the instructions at Download & Install. You may do this directly on Edison if it connects to internet. Or you may download to a PC connected internet and then copy the entire iot-sol directory to Edison via sdcard or scp.

Start IoT SOL

Then you will see

 cd iot-sol
 ./start_mock_demo.sh

Then you will see this shown on the screen, you may read the Advanced Topics in the documentation for more details

start mock hub_b ...
